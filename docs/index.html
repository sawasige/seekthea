Seekthea

ウェブ全体のトレンドを、あなたの手に。

ニュース・テック・ソーシャルのトレンドをまとめて収集。Apple IntelligenceのAI要約で、記事のポイントが一瞬でわかる。

自分だけのトレンドフィードを作ろう

100以上のプリセットから好きなソースを選ぶだけ。AI要約で記事のポイントが一瞬でわかる。
RSSだから、アカウント登録もアルゴリズムの偏りもなし。あなたが選んだソースの、ありのままのトレンドを。

主な機能

🚀

すぐに始められる

14カテゴリ100以上のプリセットから興味のあるものを選ぶだけ。RSSの知識は不要です。

📖

3つの読み方

リーダー表示で広告を除去、AI要約で要点を即把握、Webで原文確認。記事ごとに切り替えられます。

🧠

使うほど賢くなる

読んだ記事やお気に入りから興味を学習。おすすめの精度が上がり、興味度スコアで記事の関連度が一目でわかります。

🔍

ソース自動発見

Google Newsのトレンドから新しいソースを自動的に発見して提案。自分では見つけられなかったソースが増えていきます。

☁️

iCloud同期

お気に入り・既読状態・ソース設定がiPhone・iPad・Mac・Vision Proで自動同期されます。

🎯

特定ソースで絞り込み

気になるソース1つだけを表示したり、不要なソースを除外して読めます。スコアの内訳画面で「なぜこの記事？」も確認できます。

選ぶだけで、すぐ始まる

RSSリーダーはフィードURLを登録するのがハードル。Seektheaでは14カテゴリのプリセットから興味のあるものを選ぶだけで、100以上のソースからニュースが届き始めます。

さらに、Google Newsのトレンドを監視して新しいソースを自動発見。自分では見つけられなかったサイトが勝手に増えていきます。

3つの読み方で、記事に集中

リーダー — 広告やナビゲーションを除外して本文だけを表示。記事の内容に集中できます。

AI要約 — Apple Intelligenceが記事を要約。長い記事でも要点がすぐわかります。英語の記事も日本語で要約されることがあります。

Web — 元のサイトをそのまま表示。原文を確認したいときに。

使うほど、あなた好みに

読んだ記事やお気に入りから興味を自動で学習。「おすすめ」モードでは記事ごとに0〜100%の興味度スコアが表示され、関連度が一目でわかります。

設定画面では、アプリが学習した興味傾向をランキングで確認できます。自分がどんなトピックをよく読んでいるか、意外な発見があるかもしれません。
